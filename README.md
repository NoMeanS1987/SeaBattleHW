# Морской бой (Battleship)

Консольная реализация игры «Морской бой» на Python с игрой против ИИ.

## Возможности

- Игровое поле 6×6
- Ручная расстановка кораблей игроком
- Автоматическая расстановка кораблей ИИ
- Статусы кораблей: цел / ранен / убит
- Валидация ходов и размещения кораблей

## Корабли

| Тип | Размер | Кол-во |
|-----|--------|--------|
| 3-палубный | 3 | 1 |
| 2-палубный | 2 | 2 |
| 1-палубный | 1 | 3 |

## Запуск
```bash
python battleship.py
```

## Технологии

- Python 3
- ООП: классы `Ship`, `Board`, `AIPlayer`, `Game`
